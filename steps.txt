In the context of the example workflow:
1. Create IAM role for all lambdas
  - Create actual role (exampleWorkflowLambdaRole)
  - Attach AWSLambdaBasicExecutionRole and AWSLambdaRole to exampleWorkflowLambdaRole
2. Create IAM role for step function
  - Create actual role (exampleWorkflowStepFunctionsRole)
  - Attach AWSXRayDaemonWriteAccess and AWSLambdaRole to exampleWorkflowStepFunctionsRole
3. Create Lambdas
  - 4 lambdas
    - Provisioner
    - Requester
    - Manager
    - Validator
  - For each in lambdas/*, zip each file
  - For each lambda, create lambda
    - Role: exampleWorkflowLambdaRole    
    - runtime: nodejs12.x
    - handler: $filename.handler
    - function name: $filename
    - code: zip file: lambdas/$filename.zip
4. Create state machine
  - Role: exampleWorkflowStepFunctionsRole
  - Definition: read asl json file
  - Name: name of the json file
5. Invoke?

note: role arns are required when creating functions and state machines
